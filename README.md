# LLM Log Message Rewriter (LLM 日志信息改写器)

这是一个基于大语言模型（LLM）的高性能、工业级日志改写工具。它专门设计用于在保持 C/C++ 格式占位符（如 `%s`, `%d`, `%zu` 等）严格一致的前提下，对大量的英文日志文本进行同义改写和润色。

## 🚀 核心功能

- **格式安全保证**：自动识别并严格保护 C/C++ 格式化占位符，确保改写后的字符串在代码中可以直接替换而不会导致崩溃或乱码。
- **高性能并发处理**：内置线程池调度器，支持多文件并发处理，轻松应对数千行日志的改写任务。
- **智能去重降本**：在处理前自动进行行去重，显著降低 LLM API 的调用成本。
- **多模型支持**：兼容 OpenAI 格式接口，完美支持 DeepSeek（云端）和 Qwen（本地 Llama/Ollama）等主流模型。
- **工业级输出**：提供标准 JSON 和 Python `r-string` 字典两种输出格式，方便直接嵌入工程代码。

## 💼 业务价值 (Business Value)

1.  **极大地提升研发效率**：
    在大型项目（如重构、合规检查或品牌化适配）中，手动改写成千上万条日志不仅耗时且极其乏味。本工具将原本需要数天的重复性劳动缩短至分钟级。
2.  **确保技术一致性**：
    人工改写极易遗漏或写错 `%zu`、`%p` 等格式符，导致程序运行时崩溃。本工具通过正则校验和 LLM 强约束提示词，确保了改写的 100% 技术准确性。
3.  **灵活的部署方案**：
    支持本地 LLM 接入，解决了企业对敏感日志数据外流到公有云的合规性担忧。

## 💰 商业价值 (Commercial Value)

1.  **显著降低运营成本**：
    通过内置的去重算法和对本地低成本模型（如 Qwen3.5:27b）的支持，在大规模改写任务中可节省 90% 以上的 API 费用。
2.  **提升软件产品质量**：
    统一、专业且易读的日志信息能显著提升最终用户对软件专业度的感知，同时高质量的日志也有助于售后支持团队更快速地定位问题。
3.  **知识产权与合规避险**：
    在处理第三方开源代码或进行闭源产品开发时，通过同义改写可以有效调整文本表达方式，符合特定的品牌规范或合规要求，降低潜在的文本侵权风险。
4.  **加速国际化进程**：
    为日志的本地化和多语言风格统一提供了标准化的自动化基石。

## 🛠️ 快速开始

### 环境准备
```bash
pip install requests
```

### 快速配置
在 `llm_msg_rewriter.py` 中配置您的 API 信息：
```python
# 配置 1: 云端模型 (如 DeepSeek)
deepseek_config = LLMConfig(
    api_key="your_api_key_here",
    base_url="https://api.deepseek.com/v1/chat/completions",
    model="deepseek-chat"
)

# 配置 2: 本地私有化模型
local_llm_config = LLMConfig(
    api_key=None,
    base_url="http://192.168.x.x:8000/v1/chat/completions",
    model="qwen3.5:27b"
)
```

### 运行
```bash
python llm_msg_rewriter.py
```
