# Contributing to LLM Log Message Rewriter

感谢你的贡献！🎉

## 如何贡献

1. 搜索 [Issues](https://github.com/huajielong/msg_rewriter/issues) 确认问题
2. Fork 本仓库
3. 创建分支：`git checkout -b feat/your-feature`
4. 提交：`git commit -m "feat: your feature"`
5. 推送：`git push origin feat/your-feature`
6. 提交 Pull Request

### 规范

- 遵循 [PEP 8](https://www.python.org/dev/peps/pep-0008/)
- [Conventional Commits](https://www.conventionalcommits.org/) 格式
